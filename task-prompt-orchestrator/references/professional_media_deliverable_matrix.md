# Professional Media and Deliverable Matrix

Use this when the user request is best routed by the output form rather than only by industry. Combine this matrix with the P-code workflow and, when relevant, an industry scenario code.

Selection rule:

1. Identify the final artifact: image, video, audio, brand design, document, slide deck, spreadsheet, web page, communication, automation, form, or report.
2. Select the closest deliverable code below.
3. Use the row to shape the prompt packet: required inputs, production steps, output format, and quality checks.
4. Use specialized plugins/skills when available for the actual artifact creation.
