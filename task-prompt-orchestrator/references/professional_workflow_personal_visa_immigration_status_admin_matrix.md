# Professional Workflow Personal Visa Immigration Status Admin Matrix

Use this matrix when a workflow must handle personal visa, immigration status, residence permit, entry authorization, consular appointment, biometrics, immigration forms, evidence packets, work authorization, student or exchange status, dependent or family applications, status renewal, extension, change, travel re-entry planning, or recurring immigration admin review.

Use this for personal organization, official-source checklists, records, deadline tracking, appointment preparation, communication drafts, and professional handoff packets. Do not use it to provide legal immigration advice, select an immigration strategy, determine eligibility, interpret statutes, guarantee approval, advise misrepresentation, replace immigration counsel, government agencies, schools, employers, consulates, carriers, border officers, or emergency services.
