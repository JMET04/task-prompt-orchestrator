# Professional Workflow College Admissions Financial Aid Student Admin Matrix

Use this matrix when a workflow must handle college, university, trade-school, community college, graduate-school, or professional-program applications; program research; admissions requirements; application portals; essays or statements; recommendation letters; transcripts and test records; scholarships; financial aid forms; student loans; campus visits or interviews; admission decisions; enrollment deposits; housing, orientation, registration, or recurring admissions-cycle review.

Use this for applicant organization, deadline tracking, records, communication drafts, application packets, decision-support checklists, and official-office handoff preparation. Do not use it to guarantee admission or aid, fabricate credentials, ghostwrite dishonestly, provide financial/legal/tax/immigration advice, choose a school as a fiduciary, or replace school admissions offices, financial-aid offices, counselors, recommenders, testing agencies, lenders, immigration counsel, or official application portals.

## COLSCP - College Application Scope

| Code | Workflow question | Workflow use |
|---|---|---|
| COLSCP01 | What admissions cycle or student stage is in scope? | Define high school, transfer, adult learner, graduate, professional, certificate, trade school, international, or returning-student application. |
| COLSCP02 | What applicant profile is being organized? | Capture academic background, interests, constraints, citizenship/residency context, budget, location, support needs, and privacy boundaries. |
| COLSCP03 | What application goal matters? | Clarify exploration, shortlist, application completion, affordability comparison, scholarship search, enrollment decision, or deferral planning. |
| COLSCP04 | What schools or programs are being considered? | Track institution, program, degree, location, modality, selectivity, cost, deadlines, and application platform. |
| COLSCP05 | What deadline types exist? | Identify early action, early decision, regular, rolling, priority scholarship, financial aid, housing, deposit, and orientation deadlines. |
| COLSCP06 | What official sources must be used? | Prefer school admissions pages, financial-aid office pages, official portals, testing agencies, and government aid sources. |
| COLSCP07 | What sensitive data is present? | Protect grades, test scores, IDs, family finances, tax documents, immigration status, disability accommodations, and essays. |
| COLSCP08 | Who participates in the process? | Map applicant, parent/guardian, counselor, recommender, admissions officer, financial-aid office, coach, mentor, or advisor. |
| COLSCP09 | What should not be decided here? | Do not guarantee admission, determine eligibility, falsify materials, choose binding commitments, or interpret aid law. |
| COLSCP10 | What scope artifact should be produced? | Create admissions brief, school list, deadline map, privacy note, or application project plan. |
