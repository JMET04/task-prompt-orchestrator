# Professional Workflow Personal Rewards Points Miles Loyalty Admin Matrix

Use this matrix when a workflow must handle personal rewards, points, miles, loyalty accounts, airline miles, hotel points, credit card rewards, retail rewards, dining rewards, fuel points, cashback records, award travel planning records, transfer partners, point expiration, elite status, family pooling, promotions, reward redemption, reward disputes, account security, or recurring rewards review.

Use this for personal organization, account inventory, balance tracking, expiration reminders, official-program source checks, redemption comparison records, travel handoff packets, tax-prep records where needed, and security hygiene. Do not use it to provide financial advice, credit-card advice, investment advice, tax advice, travel-agent services, legal advice, guarantee award availability, optimize spending as a fiduciary, or replace program terms, banks, airlines, hotels, tax preparers, travel advisors, or consumer regulators.
