# Professional Workflow Daily Commute Public Transit Rideshare Parking Matrix

Use this matrix when a workflow must handle daily commuting, route planning, commute schedules, public transit, fare passes, rideshare, taxi, parking, walking, biking, micromobility, commute weather, accessibility, commute safety, delays, disruptions, or commuting review.

Use this for personal planning, coordination, budgeting, routines, records, and contingency planning. Do not use it to replace traffic law advice, emergency transport, official transit operations, law enforcement, professional travel security planning, medical mobility assessment, employer labor/legal advice, or licensed transportation-provider decisions.

## COMSCP - Commute Scope

| Code | Workflow question | Workflow use |
|---|---|---|
| COMSCP01 | What commute is being planned? | Define origin, destination, purpose, regularity, required arrival time, and return needs. |
| COMSCP02 | Who is commuting? | Capture solo commuter, child, elder, caregiver, visitor, carpool, coworker, or mobility-support context. |
| COMSCP03 | What mode mix is possible? | List car, transit, rideshare, taxi, walking, biking, scooter, park-and-ride, shuttle, or hybrid options. |
| COMSCP04 | What time window matters? | Note departure range, arrival deadline, peak periods, transfer windows, school/work start times, and curfews. |
| COMSCP05 | What constraints shape the commute? | Include cost, safety, reliability, accessibility, weather, luggage, dress code, childcare, pets, or parking limits. |
| COMSCP06 | What information sources are trusted? | Use official transit apps, maps, parking apps, employer alerts, school notices, weather, and local advisories. |
| COMSCP07 | What recurring pattern exists? | Separate daily routine, occasional office day, appointment trip, school pickup, night shift, or seasonal commute. |
| COMSCP08 | What personal boundary applies? | Avoid legal, emergency, medical mobility, official transit operations, or professional driver-safety judgments. |
| COMSCP09 | What success criteria define a good commute? | Set acceptable arrival buffer, cost, comfort, stress, safety, steps, transfers, and backup readiness. |
| COMSCP10 | What commute artifact should be produced? | Create commute brief, route comparison, daily checklist, or recurring commute plan. |
