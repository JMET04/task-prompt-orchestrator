# Professional Workflow Personal Style Clothing Grooming Laundry Matrix

Use this matrix when a workflow must manage personal style, clothing, wardrobe inventory, outfits, fit, grooming routines, laundry, fabric care, seasonal clothing transitions, shopping, alterations, dress codes, visual presentation, or style review.

This supports organization, decision-making, communication, and care routines. It does not replace medical dermatology, cosmetic procedure advice, employment-law advice, safety PPE standards, cultural/religious authority, or professional styling services.

## Personal Style Scope

| Code | Workflow question | Workflow use |
|---|---|---|
| STYLESC01 | What style workflow is needed? | Classify wardrobe audit, outfit planning, grooming, laundry, care, shopping, alterations, dress code, packing, or review. |
| STYLESC02 | Who is the clothing or grooming plan for? | Identify adult, child, elder, worker, student, performer, traveler, caregiver-supported person, or group. |
| STYLESC03 | What context must the appearance serve? | Capture work, school, interview, event, daily routine, travel, climate, culture, faith, accessibility, or safety. |
| STYLESC04 | What constraints matter? | Note budget, body comfort, mobility, sensory needs, climate, laundry access, time, storage, and dress rules. |
| STYLESC05 | What identity or comfort goal matters? | Define confidence, modesty, professionalism, ease, gender expression, cultural fit, practicality, or low maintenance. |
| STYLESC06 | What boundary applies? | Separate style organization from medical advice, cosmetic treatment, legal discrimination claims, PPE compliance, and professional styling. |
| STYLESC07 | What existing assets should be used? | Prefer current wardrobe, accessories, grooming tools, uniforms, care products, photos, and trusted outfit formulas. |
| STYLESC08 | What risk should be avoided? | Avoid unsafe shoes, weather mismatch, dress-code breach, fabric damage, allergic products, or unaffordable shopping. |
| STYLESC09 | What decision cadence fits? | Choose daily, weekly, seasonal, event-based, travel-based, or life-transition review. |
| STYLESC10 | What scope artifact is required? | Produce style brief, wardrobe goal, constraint list, outfit context map, or care routine plan. |
