# Professional Workflow Strategy Portfolio and Roadmap Matrix

Use this when a workflow must decide between many opportunities, projects, bets, initiatives, roadmap items, investments, or strategic directions. Strategy portfolio rules define strategic intent, portfolio framing, roadmap sequencing, objective/key result links, initiative shaping, prioritization, funding, capacity allocation, scenarios, governance, portfolio risk, and review cadence.

Selection rule:

1. Identify the strategic decision or portfolio being managed.
2. Select the closest strategy/portfolio code.
3. Add strategic objective, candidate set, criteria, time horizon, owner, constraints, tradeoffs, metrics, and review cadence to the prompt packet.
4. Do not prioritize a portfolio without explicit criteria and a decision owner.
