# Professional Workflow Personal Photos Media Memory Archive Matrix

Use this matrix when a workflow must manage personal photos, videos, screenshots, family albums, phone camera rolls, media import, deduplication, album organization, backup, cloud storage, privacy, sharing, printing, event memories, migration, or long-term memory archives. Use this for personal organization, preservation, sharing, and privacy hygiene, not to replace forensic evidence handling, professional digital asset management, legal discovery, copyright clearance, cybersecurity incident response, or therapeutic memory work.

## PHOSCP - Photo Media Scope

| Code | Workflow question | Workflow use |
|---|---|---|
| PHOSCP01 | What media collection is in scope? | Define phone photos, videos, screenshots, scans, family archive, event album, camera card, or cloud library. |
| PHOSCP02 | Why is organization needed now? | Link to full storage, device change, family event, travel, memorial, sharing need, backup risk, or clutter. |
| PHOSCP03 | Who owns or appears in the media? | Identify self, family, children, friends, guests, clients if personal, pets, or deceased relatives. |
| PHOSCP04 | What outcome is desired? | Choose clean camera roll, albums, backup, shared gallery, printed book, migration, archive, or deletion plan. |
| PHOSCP05 | What sensitivity applies? | Note children, location, IDs, medical info, private home, intimate images, grief, or relationship-sensitive media. |
| PHOSCP06 | What should be excluded? | Exclude forensic chains, legal discovery, copyright clearance, professional DAM, cybersecurity incidents, or therapy. |
| PHOSCP07 | What time range matters? | Define today, trip, year, childhood archive, old phone, family history, or entire library. |
| PHOSCP08 | What devices and services are involved? | Map phones, tablets, cameras, computers, drives, cloud accounts, messaging apps, and social platforms. |
| PHOSCP09 | What record should be kept? | Track source, import date, backup status, shared albums, deletion decisions, and archive location. |
| PHOSCP10 | What scope artifact is required? | Produce photo project brief, source map, archive plan, or album checklist. |

| Code | Workflow question | Workflow use |
|---|---|---|
| PHODED01 | What counts as a duplicate? | Distinguish exact duplicate, burst shot, edited copy, screenshot repeat, thumbnail, export, or near-duplicate. |

| PHODED05 | What cleanup rule applies to screenshots? | Separate receipts, references, memes, documents, recipes, and temporary screenshots before deleting. |
| PHODED06 | What batch size is manageable? | Process by date, event, album, source, duplicate cluster, or storage-heavy videos. |
| PHODED07 | What tool can help? | Use built-in duplicate finder, file manager, photo app, script, or visual review depending on risk. |
| PHODED08 | What deletion recovery exists? | Confirm recycle bin, recently deleted folder, backup, export, and undo window before permanent delete. |
| PHODED09 | What cleanup metric matters? | Track storage saved, duplicates removed, reviewed items, or high-value favorites marked. |
| PHODED10 | What dedupe artifact is required? | Produce cleanup checklist, duplicate policy, deletion log, or review queue. |

| PHOPRI09 | What boundary protects relationships? | Avoid surprise posting, embarrassing photos, grief-sensitive sharing, or ex-partner conflict. |
| PHOPRI10 | What privacy artifact is required? | Produce privacy checklist, consent note, safe-sharing rule, or sensitive album map. |
