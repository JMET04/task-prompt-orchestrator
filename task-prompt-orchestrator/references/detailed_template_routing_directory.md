# Detailed Template Routing Directory

Use this only after the low-cost activation front door has selected a need for exact detailed routing, or when maintaining/auditing the complete routing map. Prefer `ai_full_domain_activation_directory.md` for normal AI all-domain routing and `professional_workflow_dimension_index.md` for broad coverage audits.

## Detailed Routes
- AI full-domain activation directory: read `references/ai_full_domain_activation_directory.md` first when the user asks for AI all-domain skills/workflows, beginner-friendly AI control, prompt/workflow routing, low-cost activation, directory optimization, capability selection, or choosing the smallest workflow/prompt area to load. Use it as the front door before reading the full dimension index or many workflow matrices.
- Professional problem workflow routing: read `references/professional_workflow_library.md` when the user asks for a professional output, a best-practice process, or a domain task such as research, coding, product, marketing, sales, design, image2, data, legal, finance, compliance, QA, operations, education, or strategy.
