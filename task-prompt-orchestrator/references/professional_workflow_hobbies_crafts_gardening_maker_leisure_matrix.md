# Professional Workflow Hobbies Crafts Gardening Maker Leisure Matrix

Use this matrix when a workflow must handle hobbies, crafts, maker projects, DIY creative projects, gardening, collecting, clubs, classes, amateur performance, home workshop or studio setup, tools, materials, skill practice, project planning, safety, budget, sharing or selling small outputs, or hobby review. This matrix supports personal and amateur leisure projects; it does not replace professional construction, electrical, safety engineering, business, legal, or regulated production advice.
