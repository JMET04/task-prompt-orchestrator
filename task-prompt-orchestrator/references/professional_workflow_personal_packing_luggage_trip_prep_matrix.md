# Professional Workflow Personal Packing Luggage Trip Prep Matrix

Use this matrix when a workflow must manage personal packing, luggage, day bags, travel documents, clothing, toiletries, electronics, medications or health supplies, snacks, children's items, pet items, security rules, lost-item prevention, return unpacking, or packing review for travel, commuting, school, events, hospital visits, gym, or overnight stays. Use this for organization and preparedness, not to replace medical advice, airline/security authority rules, customs/legal advice, emergency services, or professional expedition planning.

## PACKSC - Packing Scope

| Code | Workflow question | Workflow use |
|---|---|---|
| PACKSC01 | What trip or outing is being packed for? | Define flight, road trip, commute, gym, school, hospital visit, overnight stay, event, hike, or family visit. |
| PACKSC02 | Why does packing need planning? | Link to deadline, weather, children, medication, formal event, long travel, limited baggage, or anxiety reduction. |
| PACKSC03 | Who is being packed for? | Identify self, partner, child, elder, pet, household, group, or person needing support. |
| PACKSC04 | What bag types are allowed or desired? | Note carry-on, checked bag, backpack, purse, diaper bag, gym bag, garment bag, or day pack. |
| PACKSC05 | What constraints apply? | Capture weight, size, weather, dress code, laundry access, security rules, mobility, budget, and storage. |
| PACKSC06 | What should be excluded? | Exclude medical decisions, security authority interpretation, customs/legal advice, and emergency expedition planning. |
| PACKSC07 | What priority applies? | Prioritize documents, medication, keys, wallet, phone, chargers, weather gear, and event-critical items. |
| PACKSC08 | What itinerary drives packing? | Map days, nights, activities, climate, lodging, transport, laundry, and special occasions. |
| PACKSC09 | What record should be kept? | Track packing list, bag contents, borrowed items, prescriptions, serial numbers, and return checklist. |
| PACKSC10 | What scope artifact is required? | Produce packing brief, bag plan, essential checklist, or itinerary-to-items map. |
