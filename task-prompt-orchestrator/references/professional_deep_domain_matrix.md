# Professional Deep Domain Matrix

Use this when the user request sits in a specialized professional domain that needs more than a generic workflow: research, patents, standards, government, procurement, manufacturing, supply chain, clinical/life sciences, construction, energy, real estate, or international/localization work.

Selection rule:

1. Select the normal P-code, industry code, deliverable code, and high-risk gate first.
2. If the request names a deep domain below, select the closest domain code.
3. Combine the row's domain inputs and checks into the prompt packet.
4. Treat these rows as professional scoping aids, not a replacement for licensed expert review.
