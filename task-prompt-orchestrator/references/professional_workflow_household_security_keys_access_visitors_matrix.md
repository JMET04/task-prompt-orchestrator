# Professional Workflow Household Security Keys Access Visitors Matrix

Use this matrix when a workflow must handle household access control, keys, locks, smart locks, spare keys, visitor access, contractors, guests, door codes, cameras, alarms, garage access, lockouts, access privacy, access incidents, or recurring household security review.

Use this for personal household organization, communication, and recordkeeping. Do not use it to replace emergency services, law enforcement, legal advice, licensed locksmith/security installation, cybersecurity incident response, stalking/harassment support, landlord/HOA authority, or insurance coverage decisions.

|---|---|---|

| HSECSC06 | What urgency applies? | Mark immediate lockout, suspected compromise, move-in, roommate change, guest stay, repair visit, or routine audit. |
